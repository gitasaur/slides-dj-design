### Declaring and using variables

<CH.Spotlight class="prose-p:text-xs">

  Declaring and using variables

  ```js app.js
  let message = "Hello, JavaScript!";
  console.log(message);  // logs "Hello, JavaScript!"
  ```

  ---

  Defining and calling functions

  ```js app.js focus=3:6
  let message = "Hello, JavaScript!";

  function greet(name) {
    return "Hello, " + name + "!";
  }
  console.log(greet("World"));  // logs "Hello, World!"
  ```
